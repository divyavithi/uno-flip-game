# Uno Flip Game

UNO Flip is a fast-paced card game with a twist: the deck has a light and a dark side, and players can flip the deck to change the rules and card effects.

## Features

- Full implementation of UNO Flip rules (light/dark sides, special cards, flipping)
- Play against human or bot players
- Autosave and export game state

## Getting Started

### Prerequisites
- Java JDK 20.0.2 or higher

### Run
1. **Clone the repository:**
	```sh
	git clone https://github.com/divyavithi/uno-flip-game.git
	```
2. **Run the game:**
	For best experience, open the project in IntelliJ IDEA and run the `LaunchGUI` file directly.
